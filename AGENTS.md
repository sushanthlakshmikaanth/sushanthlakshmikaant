## Project Summary
An ultra-minimalist, high-end portfolio for Sushanth Lakshmikaanth, a Founder & Developer. The site emphasizes executive authority through restraint, sophisticated typography, and strategic whitespace.

## Tech Stack
- Framework: Next.js 15+ (App Router, Turbopack)
- Styling: Tailwind CSS 4
- Animations: Framer Motion
- Icons: Lucide React
- Components: Custom-built high-end UI components

## Architecture
- `src/app`: Routes and page layouts
- `src/components`: Domain-specific sections (Hero, Work, About, etc.)
- `src/components/ui`: Reusable primitive components
- `src/lib`: Utility functions and global state

## User Preferences
- **Founder Persona**: Authority communicated through technical depth and strategic vision.
- **Ultra-Minimalist Aesthetic**: Pure black backgrounds, grayscale palette, and high-quality typography (Geist Sans).
- **Executive Restraint**: Removing unnecessary identifiers (names/titles) from global navigation to focus on content.
- **Sophisticated Motion**: Slow, cinematic reveals and purposeful micro-interactions that feel expensive and deliberate.

## Project Guidelines
- **Typography First**: Layouts are driven by typographic hierarchy and whitespace rather than decorative elements.
- **Professionalism over Flair**: Avoid loud gradients, heavy glassmorphism, or trend-focused "AI slop" aesthetics.
- **Functional Clarity**: Every element must serve a clear purpose in communicating the founder's thesis or work.

## Common Patterns
- **Monochromatic UI**: Using shades of zinc and white on black to create a timeless professional look.
- **Typographic Anchors**: Large, medium-weight headlines paired with small, tracked-out uppercase labels.
- **Minimalist CTAs**: Text-based CTAs with subtle arrow indicators rather than heavy buttons.
